1 — What is OmniGene Studio?

OmniGene Studio is an advanced desktop workspace for spatial transcriptomics, molecular docking, and signaling cascade kinetic simulations. Designed for molecular biologists, pharmacology students, and clinical researchers, it translates raw genome data (VCF files or gene listings) into interactive 3D visualizations, correlation metrics, and pathway-level flux kinetics.

2 — System Architecture

The platform operates as a secure, local-first Electron wrapper bundling a React+Zustand web client and an asynchronous FastAPI scientific calculation engine.

Figure 2.1: Architectural diagram showing local-first decoupled client-backend interface and fallback paths.

3 — Installation & Setup

The installation is distributed as a single executable. Run the installer to unpack all libraries, binaries, and local dataset fragments.

  1. Execute the setup executable on Windows 10/11 x64 systems.
  2. Launch the app via the Start Menu shortcut.
  3. On first run, the local server initializes database handles on port 8000. This requires up to 10 seconds.

5 — Molecular Docking & ODE Fallback

The docking solver executes ligand binding calculations against targets in the local WebGL viewer.

Zero-Configuration Fallback: If the local machine lacks Vina binaries, the application automatically catches the exception and routes the job to the **Omnigene Docking Engine (ODE)** which runs pure-Python matrix searches on CPU or GPU.

V2000 SDF Parsing Algorithm

6 — Statistical GSEA & Configurable Backgrounds

The Hypergeometric survival solver computes signaling over-representation stats.

Formula Comparison

Standard GSEA Survival EquationImplemented math.comb Python Code
$$P(X \ge k) = \sum_{i=k}^{\min(n, K)} \frac{\binom{K}{i} \binom{N-K}{n-i}}{\binom{N}{n}}$$ 
for i in range(k, min(n, K) + 1):
    num = math.comb(K, i) * math.comb(N - K, n - i)
    denom = math.comb(N, n)
    p_sum += num / denom
Failsafe Denominator Controls: Denominator evaluations check whether N choose n yields zero (e.g. if drawing count exceeds background population). If zero is returned, the solver raises a loud ValueError instead of silently skipping terms, preventing biased p-values.

7 — Async Robustness & Race Checks

To prevent rapid gene switches from overwriting client stores with obsolete requests, every store async action contains strict identity matches: 

if (get().selectedGene !== gene.toUpperCase().trim()) return;
set({ deconvolutionData: data });

If the user selects a new gene in the ribbon while older promises are resolving, the resolved payload is discarded rather than triggering state updates.

8 — Security Limits & Uploads

To prevent memory exhaustion attacks, the VCF file parser reads incoming data spooled in chunks rather than buffering unbounded objects: 

MAX_SIZE = 10 * 1024 * 1024  # 10MB limit
file_bytes = bytearray()
while True:
    chunk = await file.read(65536)  # 64KB blocks
    ...
    if len(file_bytes) > MAX_SIZE:
        raise HTTPException(status_code=413)

9 — Node Mapping Alerts

To ensure data transparency, the UI surfaces a pulsating warning alert whenever node symbols cannot be resolved back to canonical databases, signaling that the displayed network is missing specific pathways.
